# Slime Prototype Ownership

This folder belongs to the slime agent, but the slime prototype is locked unless the user explicitly reopens slime work.

You may edit, when reopened:

- `src/liquid/**`
- `app/slime-prototype/page.tsx`
- slime-specific docs and validation images

You may read but must not edit:

- `src/vacuum/**`
- `src/experiment/**`
- shared infrastructure unless assigned a coordinator task

Keep `/slime-prototype` slime-only. Do not mount the vacuum or add gameplay pressure.
